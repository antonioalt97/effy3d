How to Add Your GLB Models

Follow these steps to add your own 3D models to this portfolio:

File Structure

project-folder/
├── index.html
├── portfolio.html
└── models/
├── character.glb
├── weapon.glb
├── environment.glb
└── creature.glb

JavaScript Syntax

// Load GLB model
const loader = new THREE.GLTFLoader();
loader.load(
'models/your-model.glb',
function(gltf) {
scene.add(gltf.scene);
}
);

Note:

Currently showing placeholder models. Replace the model loading URLs in the JavaScript section with your actual GLB file paths. Make sure your GLB files are optimized for web viewing (under 10MB recommended).
